这个纸牌游戏示例基于Cocos2d-x游戏引擎3.8.8版本构建，实现了一个简单的纸牌记忆游戏。主要功能在于用TS代码组装卡牌，拉伸动画模拟翻牌效果，点击事件的判断和处理。


This card game example is built on the Cocos2d-x game engine version 3.8.8, implementing a simple card memory game. The main functions include assembling cards using TS code, simulating card flipping effects with stretching animations, and judging and handling click events.
